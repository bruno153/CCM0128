Como o algoritmo empregado é uma variante de quicksort, e o gráfico resultante do doubling
test é característico de linear ou linearítmico (ambos são parecidos), é possível concluir
que o algoritmo é linearítmico.

	Ainda, o programa assume que existe somente uma porca correspondente a um parafuso. 
Entretanto o algoritmo funciona bem com pilha de porcas e parafusos com tamanhos idênticos, 
como exemplificado pelo intput weirdpile.txt.
